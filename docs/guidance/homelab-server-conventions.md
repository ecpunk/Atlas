# Homelab And Server Conventions

Use discovery-first operations: verify runtime state, ports, DNS, and logs before applying fixes. Treat deployment and path standards as operational defaults, and prefer resilient automation patterns over ad-hoc manual intervention.

## Why This Matters

Operational incidents are usually caused by stale assumptions. This guidance shortens time-to-triage, reduces repeat failure modes, and keeps homelab services maintainable under change.
