# Ops Map Pattern

Maintain a compact operational map for multi-service projects that captures topology, deploy sequence, known traps, and integration touchpoints. Keep it current whenever topology changes and pair it with AI memory so human and agent sessions start from the same operational context.

## Why This Matters

Most operational delay comes from re-discovering known facts. A maintained ops map turns tribal knowledge into durable run-context and cuts repetitive orientation time during incidents and handoffs.
