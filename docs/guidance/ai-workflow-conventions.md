# AI Workflow Conventions

Use Claude for ideation, architecture, and judgment-heavy diagnosis, and use Codex for implementation and execution tasks. Hand off with outcome-oriented prompts and explicit success criteria, then feed execution results back into decision-making before the next change.

## Why This Matters

Clear lane separation reduces ambiguity and prevents the wrong tool from making high-impact decisions. Structured handoffs improve throughput while preserving architectural coherence.
