# atlas-store

## Overview

Canonical Atlas definition store bootstrap for Plan 001 Phase 1.

## Workflow

Edit YAML definitions, run validation tools, run regeneration tools, review output, and commit.

## Adding a vocabulary value

Update a vocabulary YAML file under vocabularies and validate.

## Adding a project entity

Add a project YAML file under entities/projects and validate.

## Validation

Validation tooling will be added in later phases.

## Regeneration

Regeneration tooling will be added in later phases.

## Not yet implemented

- Project and Vocabulary schema models
- Additional entity schemas
- Write API
- Generation pipeline service
- Schema versioning and migrations
- Signals and agent operational flows
